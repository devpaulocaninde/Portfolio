Login-Vaporwave

Neste vídeo, demonstro um projeto de página de login no estilo vaporwave, desenvolvido utilizando HTML5, CSS3 e JavaScript. O design combina uma estética retrô, com cores vibrantes e elementos que remetem à cultura vaporwave, enquanto mantém uma interface intuitiva e funcional. O projeto é totalmente responsivo, garantindo que a experiência do usuário seja otimizada para diferentes dispositivos e tamanhos de tela. Além do design criativo, a página de login conta com validações básicas feitas em JavaScript, proporcionando uma interação dinâmica e segura.
